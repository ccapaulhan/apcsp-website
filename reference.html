Command Description Example of Usage
man This command shows you the interface showing the manual for a linux command man mkdir
cal This command shows you a calender and current date cal 08 2000
arch This command shows you the current machine hardware name arch --version
ls This command lists the contents of a directory ls Pictures
clear This command clears the terminal screen clear
last Shows a list of the last logged on users last -t today
who This command shows who is currently logged in who | more
id This command prints real user identification id -G newuser2929
date This command displays the current date date --date="two year ago"
echo This command dislays text to the interface echo warriors
